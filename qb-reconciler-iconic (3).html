Instructions

Setup Data Format Guide
1
Payroll Data File

Export from FOCUS EHR. Sheet name: Payroll Data

Column Header Status
ASource.NameRequired
JProvider NameRequired
NProvider PaymentRequired
Source.Name Format: Payroll Workbook 06.06.25.xlsm

Date in filename QB payment is ~7 days later

2
QuickBooks Data File

Export from QuickBooks. Sheet name: QB Data

Column Header Status
AContractor NameRequired
BBusiness NameRequired
EDateRequired
HPaid AmountRequired

QB payment dates should be ~7 days after Payroll file dates (3 day tolerance)

3
Name Mapping File (Optional)

CSV or Excel file to map contractor names to business names

Column Header Status
AContractor NameRequired
BBusiness NameRequired
Example: "John Smith" "Smith Consulting LLC"
4
Quick Start
  1. Upload your Payroll and QB data files
  2. Optionally add a name mapping file
  3. Click "Validate" to check file formats
  4. Click "Run Reconciliation"
  5. Review discrepancies in the Dashboard

All data is processed locally in your browser. Nothing is uploaded to any server.

Logo Format Guidelines
Best practices for optimal display
Recommended Formats
  • PNG - Best for logos with transparency
  • SVG - Vector format, scales perfectly
  • WebP - Modern format, small file size
SpecificationRecommendation
DimensionsSquare (1:1 ratio) - e.g., 256256px
Min Size6464 pixels
Max File Size500 KB
BackgroundTransparent (PNG/SVG) preferred
Color SpacesRGB
Tips for Best Results
  • Use a simple, clean logo that works at small sizes
  • Avoid text-heavy logos (text may be illegible)
  • Test with both dark and light themes
  • Logo is stored in browser localStorage
